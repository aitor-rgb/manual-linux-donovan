Manual de Comandos Linux

SORT

Ordena líneas de texto.

Opciones

  • -r
  • -n
  • -k

Ejemplos

sort archivo.txt
sort -r archivo.txt
sort -n numeros.txt
sort -k 2 archivo.txt

CUT

Extrae columnas de texto de una entrada.

Opciones

  • -d
  • -f

Ejemplos

cut -d ":" -f 1 /etc/passwd
cut -d "," -f 2 archivo.csv

HEAD y TAIL

Muestran el inicio o el final de un archivo.

Opciones

  • head -n
  • tail -n
  • tail -f

Ejemplos

head -n 10 archivo.txt
tail -n 20 archivo.txt
tail -f /var/log/syslog

FIND

Busca archivos y directorios según distintos criterios.

Opciones

  • -name
  • -perm
  • -mtime
  • -exec
  • -size

Ejemplos

find . -name "*.txt"
find /home -perm 644
find /var/log -mtime -7
find . -name "*.log" -exec rm {} \;
find . -size +10M
